EPHRS Bracket EXE

1. Double-click EPHRS_Bracket.exe.
2. The app opens in its own browser-app window.
3. Use Save to persist tournament-state.json next to the EXE.
4. Export as PNG/PDF downloads the file through the browser window.

Notes:
- The EXE keeps all app data beside itself.
- The bundled export flow was tested end to end with PNG and PDF automation.
